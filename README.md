# Short_url_project

- link short url : https://short-url-b7le.onrender.com/

- วิธีการติดตั้ง
    - clone ไปยังโฟลเดอร์
    - เปิด terminal ขึ้นมา แล้วพิมพ์ ```npm install```
    - จากนั้นพิมพ์ ```npm run dev```
    - เปิด terminal ขึ้นมาอีก tab
    - เข้าไปยังโฟล์เดอร์ client ใช้คำสั่ง ```cd .\client\```
    - พิมพ์ ```npm install```
    - จากนั้นพิมพ์ ```npm run dev```
